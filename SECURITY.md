# Security Policy

## Supported Versions

Only the latest state of the `main` branch is supported with security updates.

| Version | Supported |
| ------- | --------- |
| `main`  | ✅ |
| All other versions, branches, forks, or snapshots | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly and do not open a public issue.

You can report vulnerabilities by contacting the maintainer directly through GitHub or via the contact information listed on the website.

Please include the following information where possible:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested mitigation or fix (optional)

You can expect an initial response within a reasonable timeframe. Valid reports will be investigated and, when confirmed, addressed in the `main` branch.

Please do not publicly disclose security issues until they have been reviewed and resolved where applicable.
